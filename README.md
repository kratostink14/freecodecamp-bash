# Велопрокат

Интерактивная система управления велопрокатом через командную строку на Bash и PostgreSQL.  
Создано в рамках сертификации [freeCodeCamp Relational Database](https://www.freecodecamp.org/learn/relational-database/).

## Функционал

- Просмотр доступных велосипедов по типу и размеру
- Автоматическая регистрация новых клиентов
- Аренда велосипеда с записью в базу данных
- Возврат велосипеда с обновлением доступности
- Валидация ввода и обработка ошибок

## Технологии

- **Bash** — скрипты и логика CLI
- **PostgreSQL** — хранение данных (велосипеды, клиенты, аренды)
- **psql** — запросы к базе данных из shell

## Схема базы данных

| Таблица     | Основные колонки                                    |
|-------------|-----------------------------------------------------|
| `bikes`     | bike_id, type, size, available                      |
| `customers` | customer_id, name, phone                            |
| `rentals`   | rental_id, customer_id, bike_id, date_returned      |

## Установка и запуск

**1. Восстановить базу данных**

```bash
psql --username=postgres -c "CREATE DATABASE bikes;"
psql --username=postgres bikes < bikes_dump.sql
```

**2. Запустить скрипт**

```bash
chmod +x bike-shop.sh
./bike-shop.sh
```

> Требуется установленный PostgreSQL. Пользователь `freecodecamp` должен существовать в системе.
